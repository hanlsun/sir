회원 가입 방법 안내

회원 가입 방법

1. https://aiis.snu.ac.kr/ 로 접속합니다.

2. 아래 사진에 표시된 버튼(항목이름: ‘SIR’)을 클릭하여 사업단 홈페이지에 접속합니다.

3.사업단 홈페이지 접속 후 '데이터 구축 프로그램 바로가기' 버튼을 클릭합니다.

4. ‘회원가입’ 버튼을 클릭합니다.

5.개인정보(아이디, 이메일, 이름, 휴대폰 번호, 비밀번호)를 모두 입력합니다.

6.모두 동의를 선택합니다.

7.‘회원가입’ 버튼을 눌러 완료합니다.

프로젝트 참여 방법 안내

프로젝트 참여 방법

1. 아이디와 비밀번호를 입력하고 로그인을 합니다.

2. 라벨링 또는 검수 버튼을 클릭하여 프로젝트에 참여합니다.

3. 작업 할당받기 버튼을 클릭하여 작업을 시작합니다.

급여 산정 방법 안내

라벨러 급여 산정 방법

1. 작업 통계 메뉴에서 라벨링 통계를 확인합니다.

2. 이번 달 작업 완료 건수를 확인합니다.

3. 1건당 100원으로 산정합니다.

예) 위의 예시 이미지와 같이 작업 완료 건수가 10건이라면 10*100(원) 으로 계산하여 해당 월에는 총 1000원이 산정됩니다.

검수자 급여 산정 방법

1. 작업 통계 메뉴에서 검수 통계를 확인합니다.

2. 이번 달 작업 완료 건수를 확인합니다.

3. 1건당 50원으로 산정합니다.

예) 위의 예시 이미지와 같이 작업 완료 건수가 10건이라면 10*50(원) 으로 계산하여 해당 월에는 총 500원이 산정됩니다.

라벨링 작업 안내

라벨링 작업 방법

1. 파일명(파란글씨)을 클릭합니다.

2. 다음과 같은 화면이 나오면 작업을 시작합니다.

3. 사진 속 인물이 남자면 Man 여자면 Woman 을 클릭합니다.

4. 사진 속 인물의 얼굴에 맞춰 네모박스를 그립니다.

5. 사람이 바라보고 있는 방향을 로봇 관점에서 체크합니다.

6. 사람이 느끼고 있는 감정 상태를 체크합니다.

7. 동영상을 들었을 때 들리는 말을 듣고 알맞게 입력합니다.

8. 말하는 사람의 어조에 따라 감정 상태를 체크합니다.

9. 들리는 목소리에서 다음에 해당되는 것 중 하나를 선택합니다.

10. 사람이 로봇과 대화를 하고 반응하는지 체크합니다.

검수 작업 안내

검수 작업 방법

1. 파일명(파란글씨)을 클릭합니다.

2. 다음과 같은 화면이 나오면 작업을 시작합니다.

3. 선택한 성별 태그와 네모박스가 그려진 얼굴의 성별이 일치하는지 확인합니다.

4. 네모박스가 그려진 인물을 기준으로 해당인물의 시선방향이 올바르게 체크 되었는지 확인합니다.

5. 들리는 목소리 어조를 통해 나타나는 화자의 감정상태가 체크된 내용과 일치하는지 확인합니다.

6. 영상재생 후 들리는 대화내용이 텍스트 내용과 일치하는지를 확인합니다.

7. 들리는 내용을 통해 나타나는 화자의 감정상태가 일치하는지 확인합니다.

8. 들리는 내용을 통해 나타나는 화자의 의도가 체크된 내용과 일치하는지 확인합니다.

9. 사람이 로봇과 상호작용을 하고 있는지 판단합니다.

10. 올바르게 작업이 완료된 경우 PASS를 클릭하고 잘못된 경우 FAIL을 클릭합니다.
