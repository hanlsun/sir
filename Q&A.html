FAQ

  • ❖ 스마트폰으로 작업을 할 수 있나요?

    • ➢ 스마트폰으로 진행이 불가합니다. 컴퓨터를 사용하여 주시기 바랍니다.

  • ❖ 인터넷 익스플로러에서 작업이 안돼요.

    • ➢ 크롬, 파이어폭스, 엣지 브라우저에서만 작업이 가능합니다.

  • ❖ 네모박스를 얼굴에 그릴 때 크기는 어떻게 해야하나요?

    • ➢ 얼굴 전체가 다 들어오게 네모박스를 그리면 됩니다.

  • ❖ 인물표정이 모호한 경우 어떻게 해야하나요?

    • ➢ 생각을 오래해야하거나 모호한 경우 해당없음으로 표기

  • ❖ 얼굴 방향과 눈동자 방향이 다른 경우 어떻게 해야하나요?

    • ➢ 얼굴방향과 눈동자 방향이 다른 경우 얼굴 방향을 기준으로 표기

  • ❖ 인물의 말에서 느껴지는 감정이 모호해서 선택하기 어려울 때는 어떻게 하나요?

    • ➢ 시간이 걸리고 선택하기 어려울 때는 해당없음으로 표기

  • ❖ 인물의 감정에 대해서 추가적으로 고려해야할 사항이 있나요?

    • ➢ 사진을 보고 판단할 경우는 얼굴 '표정만', 동영상을 보고 판단할 경우는 '목소리만' 으로 판단해 주시면 됩니다.

  • ❖ 질문과 요청의 구분을 어떻게 해야하나요?

    • ➢ 먼저, 질문에 대한 예시는 의문문이면서 행동을 동반하지 않는 것으로간주하면됩니다. 서울대학교AI연구원이 어디에 있나요? AI 연구원은 무엇을하는 곳인가요? 요청에 대한 예시는 행동을 동반하는 것을 요구하는 것으로보시면 됩니다. 예를 들어, AI연구원까지 데려다주시겠어요. 추가적으로 자세한내용은 도움말 및 작업 방법을 참고하시면 됩니다.

  • ❖ 다음 작업을 넘어갔는지 구분이 안가요.

    • ➢ 현 작업이 끝나고 다음 작업으로 넘어갈 시 작은 팝업창 먼저 뜹니다. 동일한 작업모습이어도 에러가 아닌 작은 팝업창이 뜨고 넘어갔다면 다음 작업을 진행한다고생각하시면 됩니다.

  • ❖ 최종 제출 버튼을 꼭 눌러야 하나요?

    • ➢ 할당된 작업을 모두 종료하고, 리스트에 미제출없이 제출이라고 표기되면 최종제출 버튼을 눌러주셔야 작업이 완료됩니다
